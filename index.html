A Brief Introduction

What is an RNA Triplet

Ribonucleic Acids (RNA) are a single chain nucleic acid sequences. To stebilize its structures, it creates base-base interactions or base pairing. Some times one base interacts or forms base pairs with more than one bases (e.g C=G=A. here the central guanine forms a pair with C and another pair with A). In such cases we call them base triplets. This web server provides a dataset of all such triplets in the RNA sets.
